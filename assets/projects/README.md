# Project Images

Place project images inside the matching numbered folder.

Recommended naming:

- `cover.webp` for the main preview image.
- `01.webp`, `02.webp`, `03.webp`, and so on for gallery images.

Recommended dimensions:

- Landscape orientation.
- Approximately 1600 x 1000 pixels.
- JPG or WebP format.
- Keep each image below 1 MB when possible.
